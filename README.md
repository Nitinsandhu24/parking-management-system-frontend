# parking-management-system-frontend
This project is a frontend application built using React for the Parking Management System. It provides an intuitive UI for users to interact with the backend services, including authentication, booking parking slots, and viewing analytics.

Tech Stack
1.React
2.JavaScript (ES6+)
3.HTML5 & CSS3
4.Axios (API calls)
5.React Router
6.Tailwind CSS

Setup
1.https://github.com/Nitinsandhu24/parking-management-system-frontend
2.cd parking-management-system-frontend
3.npm install
4.npm run dev

App will start at port http://localhost:3000
